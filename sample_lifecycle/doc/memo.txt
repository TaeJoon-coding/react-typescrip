EX）宿の場合、

勉強(関数) { 「コーディング」 } =>
昼間(関数) { 「コーディング」 } =>
勉強(関数) { 「コーディング」 } =>
修了(関数) { 「コーディング」 } =>
帰宅(関数) { 「コーディング」 } =>

関数に当たるところはプロジェクトによる。
コーディングの所は人次第。

早退の例を出すと、
勉強(関数) { 「コーディング」 } =>
昼間(関数) {} =>
勉強(関数) {} =>
修了(関数) {} =>
帰宅(関数) { 「コーディング」 } =>
のように途中のところがなくなり、速やかに終わりのコードまで実行。（SKIP）
